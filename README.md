# 下課小遊戲 / Xiake Games

免費、免下載、手機可玩的校園主題網頁小遊戲網站。

主打遊戲：`老師來了！`

## Local preview

Open `index.html` directly in a browser, or run a simple static server:

```bash
python -m http.server 8765
```

Then visit `http://127.0.0.1:8765/`.

## Production domain

https://xiakegames.com

## Deploy

This project is static HTML/CSS/JS and can be deployed to Vercel, Cloudflare Pages, Netlify, GitHub Pages, or any static host.
