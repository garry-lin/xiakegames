Contact Us

Contact Xiake Tools for bug reports, tool suggestions, privacy questions, and site feedback.

Last updated: June 17, 2026

Contact email

For bug reports, tool suggestions, privacy questions, policy questions, or general feedback, email contact@xiakegames.com. Please include the page URL, device type, browser name, and a short description of the issue when possible.

What to report

You can report a game that does not start, mobile layout problems, a tool result that looks incorrect, spelling mistakes, broken links, privacy concerns, visitor safety suggestions, or ideas for future tools.

How feedback is handled

Issues that affect core gameplay, tool accuracy, privacy pages, advertising review, or user safety are prioritized. General suggestions are collected for future updates.

Do not send sensitive data

Please do not send private lists, passwords, payment data, identification documents, or unnecessary personal information in feedback messages.

Response expectations

The site is maintained as a lightweight public project, so response time can vary. Reports about broken pages, privacy information, AdSense/ads.txt issues, or unusable tools are treated as higher priority than general feature requests.

Accessibility feedback

If a button, form, game area, contrast level, or mobile layout is difficult to use, include the device and browser in your message. This helps us reproduce the problem and improve the page for more visitors.

Security and abuse reports

If you notice a page being abused, a misleading link, an unsafe layout, or a technical issue that could affect visitors, please include the exact URL and a short reproduction note. Do not send exploit code unless it is necessary to explain the issue, and do not include private data from other people.

What makes a useful report

A useful report explains what you expected, what actually happened, and whether the issue happens every time or only sometimes. For tool pages, include a short sample input when possible. For layout issues, mention the screen size or device model. For policy or privacy questions, include the page URL so we can review the exact text you saw.

Feature suggestions

Suggestions are welcome, especially for practical tools that can run locally in the browser and help global visitors with writing, calculations, study, group activities, or everyday productivity. We prioritize ideas that are useful without collecting personal data or requiring visitors to create an account.
