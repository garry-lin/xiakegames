# 上線與 AdSense 送審檢查清單

## 網域設定
- [ ] 將整個資料夾上傳到正式網域。
- [ ] 網站可用 HTTPS 開啟。
- [ ] 如果不是使用 https://xiakegames.com，請整批取代 sitemap.xml、robots.txt、頁面中的聯絡信箱網域。
- [ ] 建立正式聯絡信箱：contact@xiakegames.com，或改成你的真實信箱。

## 頁面檢查
- [ ] 首頁可開啟。
- [ ] 遊戲頁可開啟，遊戲可正常操作。
- [ ] 攻略、玩法、FAQ、隱私權、使用條款、家長頁都可從導覽或頁尾點到。
- [ ] 沒有 404、空白頁、測試文字、製作中字樣。

## AdSense 前
- [ ] 送審前先不要放大量廣告碼。
- [ ] 若放廣告，必須在遊戲操作區外，不靠近按鈕。
- [ ] 不鼓勵、要求或暗示使用者點擊廣告。

## 手機實測
- [ ] iPhone Safari 可玩。
- [ ] Android Chrome 可玩。
- [ ] 橫向與直向排版可讀。
