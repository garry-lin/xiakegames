免費線上工具

這裡收錄適合學生、老師、寫作者與日常工作使用的小工具。工具直接在瀏覽器中操作，不需要註冊帳號，也不需要安裝 App。

為什麼在遊戲站加入工具？

下課小遊戲一開始是校園主題網頁遊戲，但學生、老師與家長在搜尋遊戲時，也常需要簡單的學習與文字工具。這些工具不走複雜功能路線，而是把常見任務做好：清楚的輸入區、即時結果、可複製輸出、手機可用，以及每個工具都有使用說明與注意事項。

工具列表

Tool

字數統計工具

統計字數、字元數、句子、段落、閱讀時間與常見關鍵字。

Tool

文字大小寫轉換

將英文文字轉成大寫、小寫、標題式、句首大寫、camelCase、snake_case 與 kebab-case。

Tool

隨機抽籤工具

貼上名單後隨機抽出一位，適合課堂點名、活動抽獎與分組暖身。

隱私提醒

多數工具會在你的瀏覽器中處理文字。請避免貼上密碼、學生個資、付款資料或其他敏感資訊。

推薦使用方式

把工具加入書籤，搭配遊戲頁與攻略頁使用；如果你是老師，可以把字數統計與隨機抽籤用在課堂活動。
